# DarksApp
An Extension to make web WhatsApp dark

## How to use:

### Firefox
- Open the `about:debugging` page
- Click `This Firefox`
- Click `Load Temporary Add-on`
- Load the `manifest.json`

### Chrome
- Open the `chrome://extensions/` page
- Enable `Developer mode`
- Click `Load unpack`
- Load the extension folder
